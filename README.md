# Simple Calculator (Java)

## Project Overview

This project is a command-line based Simple Calculator developed using Java.
The application performs basic arithmetic operations by taking user input from the console and executing the selected operation.

The project demonstrates fundamental Java programming concepts including input handling, conditional statements, and program flow control.

---

## Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division operation with division-by-zero handling
* Interactive command-line interface

---

## Technologies Used

* Java
* Scanner Class for user input
* Switch Case statements
* Command Line Interface (CLI)

## How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/YOUR-USERNAME/simple-calculator-java.git
```

### 2. Navigate to the Project Directory

```
cd simple-calculator-java
```

### 3. Compile the Program

```
javac SimpleCalculator.java
```

### 4. Run the Program

```
java SimpleCalculator
```
